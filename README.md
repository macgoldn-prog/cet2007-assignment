# Game Library and Player Manager - Maciej Goldyn (bi79fd)

**What does this project do?**

This project is a game and player manager which is able to store, add and remove games. It sorts games and players' scores/hours played and is able to search through players IDs to find the correct username.

This program utilises both binary and linear searches, as well as built in sorts.

**How do I run it?**

To run this project, you must clone this repository and run either the EXE or via VS2022.
VS2026 may have issues with this project.
